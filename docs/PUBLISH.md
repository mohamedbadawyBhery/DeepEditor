# نشر DeepEditor كهدية للبشرية

## 🎯 الهدف  
تقديم أداة تحرير كود ووعي ذكي، تعمل محلياً، بلا تعقيد، بلا إنترنت، بلا وصاية.

## 📦 طريقة النشر  
1. إنشاء حساب على [GitHub](https://github.com).
2. عمل مستودع جديد باسم `DeepEditor`.
3. رفع الملفات التالية:
   - مجلد `editor-ui`
   - مجلد `server`
   - مجلد `plugin` (للمطورين)
   - الملفات: `README.md`, `PUBLISH.md`, `start-all.bat`
4. كتابة وصف للمشروع:  
   *"DeepEditor – محرر كود وبيئة شات ذكية، مع 21 أداة نظام، تعمل عبر LM Studio"*.
5. اختيار رخصة: `MIT` أو `GPL-3.0`.

## 🔌 تثبيت البلجن في LM Studio  
1. افتح LM Studio → Plugins → Install from folder.  
2. اختر مجلد `plugin` المرفوع.  
3. تأكد من ظهور الأدوات الـ 21 في واجهة LM Studio.
